## Descrição do Projeto

Este é um **sistema web completo de gerenciamento de clientes (CRUD)** desenvolvido com **Flask + SQLite + Bootstrap 5**, como parte do **Projeto Integrador II**.

O objetivo foi **continuar e aprimorar** o projeto iniciado no **Projeto Integrador I**, aplicando:
- Melhoria contínua (Kaizen)
- Feedbacks reais de usuários
- Testes de validação
- Documentação profissional
- Versionamento com Git
